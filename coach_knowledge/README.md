# 강사 교재 (RAG)

이 폴더의 `.md` / `.txt` 파일이 AI 코치 답변의 **근거 자료**로 사용됩니다.

## 추가 방법

1. PDF·노션 내용을 마크다운으로 저장해 이 폴더에 넣기
2. 터미널에서 인덱스 갱신:

```powershell
.\venv\Scripts\python.exe coach_rag.py --force
```

3. 웹앱 재시작 (또는 다음 채팅부터 자동 반영)

## 팁

- 제목은 `# 음정 기초`처럼 첫 줄에 `#`으로 적으면 검색 결과에 잘 보입니다.
- 한 문단은 3~8문장 정도가 검색에 유리합니다.
- `COACH_RAG_ENABLED=0` 으로 RAG를 끌 수 있습니다.
