# 매일 10~15분 연습 루틴 (학원 표준)

학생 대부분은 「길게」보다 **「짧게 매일」**이 성장 곡선이 좋습니다. 아래 순서를 그대로 따라 하세요.

## 15분 풀 루틴

① **2분** — 복식호흡 4-4-8  
② **3분** — 약한 영역만 (음정/박자/호흡 중 하나) 0.5배속 구간 루프  
③ **5분** — MR 80% 속도로 한 절  
④ **3분** — 오늘 잘된 한 마디 롱톤  
⑤ **2분** — 전체 통으로 한 번 + 셀프 녹음

## 분석 점수별 우선순위

- **음정 < 70** → ②③의 시간을 음정 구간에 몰빵  
- **박자 < 70** → 메트로놈 + MR 70%  
- **호흡·음색 < 70** → 롱톤 + 복식호흡 비중 ↑

## 1주 후 재녹음 체크

- 같은 곡·같은 구간으로 다시 녹음  
- 종합 점수 + 약한 Stage 점수 비교  
- 틀린 구간 **개수**가 줄었는지 확인 (점수만 보지 말 것)

## DM으로 질문할 때 좋은 예

- 「35~40초 구간만 연습법 알려주세요」  
- 「박자 68점인데 메트로놈 BPM 몇으로 시작할까요?」  
- 「오늘 10분 루틴 짜 주세요」

구체적인 **초 구간·점수·곡**을 적을수록 선생님 답이 정확해집니다.
