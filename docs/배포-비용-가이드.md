# MVP 배포 · 비용 가이드

## 비용 요약 (초기)

| 항목 | 서비스 | 예상 비용 | 비고 |
|------|--------|-----------|------|
| 개발 도구 | Cursor, VS Code, Git | **무료** | 무료 티어 |
| 웹 배포 | Streamlit Cloud | **무료** | 트래픽↑ 시 유료 |
| AI API | OpenAI GPT-4o mini | **월 $1~5** | 사용량 과금 |

**초기 전략:** 웹 MVP 먼저 → Streamlit Cloud 무료 배포 → 사용자 반응 확인 후 DB·앱 투자

## 1. Streamlit Cloud 배포 (웹 MVP)

### 준비

1. GitHub에 `vocal-coach` 저장소 push (`.env`는 **제외**)
2. [share.streamlit.io](https://share.streamlit.io) 가입
3. New app → 저장소 선택 → Main file: `app.py`

### Secrets (GPT 사용 시)

Streamlit Cloud → Settings → Secrets:

```toml
OPENAI_API_KEY = "sk-..."
OPENAI_MODEL = "gpt-4o-mini"
```

`gpt_coach.py`가 `.env` 대신 환경변수도 읽도록 이미 구성되어 있어야 함 — 확인 필요.

### 주의

- `records/`, `clips/`는 서버 재시작 시 초기화될 수 있음 → 2단계에서 Supabase로 이전
- 분석 1회 1~3분 → Streamlit Cloud 무료 tier 타임아웃 주의

## 2. API 서버 배포 (모바일 3단계)

로컬 테스트:
```powershell
.\run_api.bat
# http://localhost:8000/docs — Swagger UI
```

배포 후보 (무료 tier):
- **Railway** / **Render** — FastAPI + uvicorn
- **Fly.io** — Docker

모바일 앱 `mobile/src/api/client.ts`의 `API_BASE_URL`을 배포 URL로 변경.

## 3. 모바일 — WebView vs Expo

| 방식 | 비용 | 난이도 | 추천 시점 |
|------|------|--------|-----------|
| **WebView** | 거의 0 | 낮음 | 지금 바로 앱처럼 보이게 |
| **Expo 네이티브** | 중간 | 중간 | 3단계, 녹음 UX 중요할 때 |

WebView: React Native `WebView`로 Streamlit URL 로드 → 스토어 제출 가능.
