## 오늘 (Step 1) — 웹에서 직접 테스트

1. **분석 대시보드** 탭 열기
2. 본인 **녹음 MP3/WAV** 업로드 (또는 "샘플로 테스트" 체크)

4. 결과 확인: 종합 점수 · 코칭 · GPT · 클립

## 이번 주 (Step 2) — Streamlit Cloud 배포 (링크 공유)

친구에게 URL 하나로 공유하려면 클라우드 배포가 필요합니다.

### 2-1. Git 설치

PowerShell (관리자):
```powershell
winget install --id Git.Git -e
```
설치 후 **PowerShell을 새로 열기**

### 2-2. GitHub 저장소 만들기

1. [github.com](https://github.com) 가입/로그인
2. **New repository** → 이름: `vocal-coach` → Create
3. 프로젝트 폴더에서:

```powershell
cd "C:\Users\chahy\OneDrive\바탕 화면\vocal-coach"
git init
git add .
git commit -m "vocal-coach web MVP"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/vocal-coach.git
git push -u origin main
```

> `.env`는 `.gitignore`에 있어 업로드되지 않습니다. (안전)

### 2-3. Streamlit Cloud 배포

1. [share.streamlit.io](https://share.streamlit.io) → GitHub 연동
2. **New app** → 저장소 `vocal-coach` → Main file: **`app.py`**
3. **Advanced settings → Secrets** 에 추가:

```toml
OPENAI_API_KEY = "sk-..."
OPENAI_MODEL = "gpt-4o-mini"
```

4. **Deploy** → 몇 분 후 `https://xxxx.streamlit.app` URL 생성

자세한 내용: [배포-비용-가이드.md](배포-비용-가이드.md)
