# 폴더 구조

```
vocal-coach/
├── README.md                 # 프로젝트 입문
├── docs/
│   ├── 기능과-사용법.md        # 상세 가이드
│   └── 폴더-구조.md            # 이 파일
│
├── analysis.py               # ★ 메인 분석 엔진 (CLI)
├── app.py                    # ★ 웹 MVP (Streamlit navigation)
├── api_server.py             # ★ 모바일 연동 REST API
├── ui/
│   ├── dashboard.py          # 분석 대시보드
│   ├── my_page.py            # 마이 페이지 (이력)
│   └── components.py         # 공통 UI
├── mobile/                   # Expo React Native (3단계)
│   ├── App.tsx
│   └── src/api/client.ts
├── .streamlit/config.toml    # 웹 테마
├── check_setup.py            # 환경 점검
│
├── action_plan.py            # 우선순위 3가지 플랜
├── progress_tracker.py       # JSON 기록·비교
├── progress_chart.py         # 성장 그래프
├── clip_exporter.py          # 구간 WAV 클립
├── mr_detect.py              # MR/반주 감지
├── dtw_compare.py            # DTW 음정 비교
├── reference_fetcher.py      # 유튜브 레퍼런스
├── gpt_coach.py              # GPT 코칭
│
├── run_check.bat             # 환경 점검
├── run_analysis.bat          # 기본 분석
├── run_record.bat            # 기록+클립+그래프
├── run_gpt.bat               # GPT 포함
├── run_web.bat               # 웹 UI
├── run_api.bat               # API 서버 (모바일)
│
├── requirements.txt
├── .env                      # OPENAI_API_KEY (비공개)
├── .env.example
│
├── sample.mp3                # 테스트용 샘플
├── test_voice.wav
├── pitch_result.png          # 최근 음정 그래프
├── analysis_report.json      # 최근 JSON (선택)
│
├── records/                  # ★ 날짜별 성장 기록 JSON
├── charts/                   # ★ growth_chart.png
├── clips/                    # ★ 집중 연습 WAV 클립
├── .cache/                   # 업로드·유튜브 임시 (자동 삭제)
│
└── venv/                     # Python 가상환경
    ├── Scripts/python.exe
    └── run_*.bat             # 상위 폴더 bat 바로가기
```

## 데이터 흐름

```
[녹음 MP3/WAV]
      ↓
 analysis.py / app.py
      ↓
 ┌────┴────┬──────────┬─────────┐
 ↓         ↓          ↓         ↓
Stage1~4  pitch_     records/  clips/
리포트    result.png  JSON      WAV
          ↓
      charts/growth_chart.png (기록 2개 이상 시 유의미)
```

## Git 제외 (.gitignore)

- `venv/`, `.env`, `.cache/`
- `records/`, `clips/`, `charts/` (개인 연습 데이터)
